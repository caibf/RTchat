## 📖 简介

RTchat (Radio Therapy) chat 是一个集成了放疗基础知识科普和互动问答的大语言模型。该模型基于 InternLM2-Math-7B 模型，通过 xtuner 微调，专门设计用于解答放疗基础知识问题。
